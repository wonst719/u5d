# u5d - Ultima V 디컴파일 프로젝트

## 소개

이 프로젝트는 현재 진행 중인 Ultima V: Warriors of Destiny 디컴파일 프로젝트입니다.

[u4-decompiled](https://github.com/ergonomy-joe/u4-decompiled) 프로젝트에서 영감을 받아, 원작 게임과 코드를 깊이 이해하기 위한 개인적인 도전으로 시작했습니다.

## 목표

- 현대적인 플랫폼에서 실행 가능한, 원작과 기능적으로 동등한 버전을 만든다.

- 원본 코드(MS-DOS 버전 1.16)와 최대한 많은 부분을 매칭한다.

원작 빌드에 사용된 링커 (PLINK86) 가 발견되지 않았기 때문에 완전한 바이너리 매칭은 아직 불가능합니다. 대신 어셈블리 수준의 시맨틱 매칭을 목표로 하고 있습니다.

필요한 경우 FM-TOWNS 등 다른 플랫폼의 디스어셈블리도 참고합니다.

## 현재 상태

### 매칭 진행도

- `ULTIMA.EXE`: 약 55% 매칭
- `*.OVL`: 약 50% 매칭

### 플레이 가능 상태

오리지널 데이터 파일을 사용해 게임이 실행되며 일부 플레이가 가능합니다. 하지만 크고 작은 버그가 아직 곳곳에 존재하며, 게임을 엔딩까지 진행할 수 없는 상태입니다.

## 빌드

### Windows 타겟

Visual Studio 2026으로 `u5win/u5win.slnx` 를 열고 빌드합니다.

### MS-DOS 타겟 (DJGPP)

DJGPP가 설치된 환경에서 `make` 를 실행합니다.
환경에 따라 `Makefile` 의 `CROSS_CC_PREFIX` 값을 수정해야 할 수 있습니다.

### MS-DOS 타겟 (오리지널 툴체인)

Microsoft C 5.1이 설치된 머신 또는 가상 머신에서 `src\build.bat` 를 실행합니다.

컴파일러 설치 위치는 `C:\MSC51` 입니다.

참고: 현재는 실행 파일 링크가 불가능합니다. 이 타겟은 디스어셈블리 매칭 목적으로만 사용됩니다.
